# 臺鐵縱貫線 列車模擬駕駛

BVE 風格的第一人稱火車駕駛模擬遊戲，以台灣縱貫線（基隆 → 新竹，25 站、46 km）車站命名。
純 HTML5 Canvas + Web Audio 打造，單一檔案、零依賴。

**線上遊玩：** https://jack1214w.github.io/taiwan-train-sim/

## 遊戲特色

- 第一人稱駕駛艙視角，偽 3D 場景（雙軌區間、側線、月台、門型架空線）
- 四種車種：
  - **區間車**（基隆 → 樹林，各站皆停，13 站）
  - **莒光號**（基隆 → 新竹，僅停大站，長途）
  - **自強號**（基隆 → 新竹，停靠 7 站，120 km/h）
  - **普悠瑪號**（基隆 → 新竹，僅停 4 站，140 km/h 傾斜式，彎道速限加成）
- 突發事件系統：前方障礙物停車等候排除、地震減速慢行（減速曲線判定）、車內緊急事件臨時停車——配合指示有獎勵分
- 真實感物理：P1–P4 推進段位、B1–B8 煞車、EB 緊急煞車、定功率區
- ATS 超速警報與自動緊急煞車（依車種有效速限）
- 三燈式閉塞號誌系統，AI 列車（對向會車、前行貨列側線避讓事件）
- 雙語車內廣播（Web Speech API）：到站廣播、緊急煞車提醒、臨時停車廣播、事件廣播
- VVVF 變頻電聯車音效
- 停車精準度＋準點率計分，S–D 評級
- 測試模式：可選任意區間出發

## 操作

| 按鍵 | 功能 |
|---|---|
| ↑ / W | 推進段位 +1（P1〜P4） |
| ↓ / S | 煞車段位 +1（B1〜B8） |
| E | 緊急煞車（EB） |
| O | 開／關車門（停準月台 ±8m 內） |
| R | 換向器（停車時可倒車修正） |
| 空白鍵 | 鳴笛 |
| 1 / 2 / 3 | 標題畫面選擇車種 |
| T | 標題畫面進入測試模式 |
